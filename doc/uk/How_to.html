Цей документ призначено для надання опису рецептів (Як зробити ...) вирішення проблем, що часто зустрічаються, та задач користувача. Рецепти вирішення задач та проблем можуть пропонуватися користувачами для розташування у цьому документі.

  • Для користувача:
  • Встановлення OpenSCADA
  • Живий диск CD/DVD/USB
  • Порушення, сигналізація та повідомлення
  • Циклічне програмування в OpenSCADA
  • Налагодження проекту OpenSCADA
  • Перенесення конфігурацій OpenSCADA з одного проекту до іншого
  • Відео "Як зробити" від Сергію Карпешу (RU), 2018 ...
  • Оновлення 0.8.0 LTS до 0.9 LTS
  • Для розробника:
  • Збірка OpenSCADA з вихідних текстів
  • Звіт про падіння OpenSCADA
  • Створення модуля OpenSCADA
