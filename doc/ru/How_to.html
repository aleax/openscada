Данный документ предназначен для предоставления описания рецептов (Как сделать ...) решения часто встречающихся проблем и задач пользователя. Рецепты решения задач и проблем могут предлагаться пользователями для помещения в этот документ.

  • Для пользователя:
  • Установка OpenSCADA
  • Живой диск CD/DVD/USB
  • Нарушения, сигнализация и уведомления
  • Циклическое программирование в OpenSCADA
  • Отладка проекта OpenSCADA
  • Перенос конфигураций OpenSCADA из одного проекта в другой
  • Для разработчика:
  • Сборка OpenSCADA из исходных текстов
  • Отчёт о падении OpenSCADA
  • Создание модуля OpenSCADA
